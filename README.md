# NLW Events Java

Este projeto é uma aplicação de gerenciamento de eventos desenvolvida em Java, utilizando Docker e MySQL.

## Tecnologias Utilizadas

- Java
- Spring Boot
- Docker
- MySQL
- Maven

1. Certifique-se de ter o Docker e o Docker Compose instalados.
